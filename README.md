edsm-dumps-model
=======

Data models for [EDSM nightly dump files](https://www.edsm.net/nightly-dumps).

## Features

* `type_hash`
    - Add derive `type_hash::TypeHash` from [type_hash](https://crates.io/crates/type_hash) to model types
